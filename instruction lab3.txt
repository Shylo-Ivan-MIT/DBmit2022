Для використання цієї програми вам необхідно використовувати Postman.

У файлі queries.js встановлюваються налаштування для підключення до бази даних у Postgres. Ви можете замінити ці значення на свої, для підключення до існуючої бази даних або ж створити нову зі значеннями, що відповідають вказаним у файлі. У базі створіть нову таблицю використовуючи скрипти з папки lab2scripts

Інструкція застосовується для програми без змінених значень:
Виконайте команду node index.js
У Postman вкажіть адресу localhost:3000.Доступні методи: GET, POST, DELETE
localhost:3000/сhamps - вивід всіх записів
localhost:3000/сhamps/id - обрати з таблиці окремий запис по значенню id, використовується для видалення

Запис нових значень виконується у  форматі json
{ "full_name": "name",
"age": 100,
gender: "male"/"female",
"team_number": 111
}
